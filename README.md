## Seções

O site é composto por três páginas:

- **Home:** Com um design simples e intuitivo os cafés (ao todo são 14) são apresentados para a seleção do cliente;
- **CheckOut:** Nessa página tenho a visualização dos produtos escolhidos, com a possibilidade de alterar o pedido como o cliente desejar. Além de receber a informação referente a entrega e o método de pagamento.;
- **Confirmação do Pedido:** Por fim uma página de confirmação, com os dados de entrega e método de pagamento escolhido, bem como a previsão de entrega;
